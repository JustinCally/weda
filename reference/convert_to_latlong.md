# Convert zone 54/55 easting/northings into lat/longs

Convert zone 54/55 easting/northings into lat/longs

## Usage

``` r
convert_to_latlong(data)
```

## Arguments

- data:

  data.frame with three columns (Easting, Northing, Zone)

## Value

data but with Easting, Northing and Zone replaced with 'Longitude' and
'Latitude'
