# Prepare transect data for upload to the database

This function takes a list of agents and prepares the data for upload to
the database.

## Usage

``` r
prepare_transect_upload(agent_list)
```

## Arguments

- agent_list:

  A list of pointblank agents (directly from transect_dq())

## Value

A list of data frames

## Examples

``` r
if (FALSE) { # \dontrun{
prepare_transect_upload(agent_list)
} # }
```
