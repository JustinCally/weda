# Name conversions between scientific and common names

A dataset containing the scientific names and common names of taxa in
the VBA

## Usage

``` r
data(vba_name_conversions)
```

## Format

A data frame with 38,587 rows and 2 variables

## Details

- taxon_id. The VBA taxon ID

- scientific_name. The scientific name in the VBA

- common_name. The common name in the VBA
