# Prepare camera trap data for upload to the database

This function takes a list of agents and prepares the data for upload to
the database.

## Usage

``` r
prepare_camtrap_upload(agent_list)
```

## Arguments

- agent_list:

  A list of pointblank agents (directly from camera_trap_dq())

## Value

A list of data frames

## Examples

``` r
if (FALSE) { # \dontrun{
prepare_upload(agent_list)
} # }
```
